# portfolio-html-css-js
Portfolio Landing Page

Description:
Responsive portfolio landing page built from a Figma design using HTML, CSS, and vanilla JavaScript. Features smooth scrolling, interactive UI elements, and form submission, focusing on clean layout and usability. Front-end project showcasing web development skills and responsive design.

Tech Stack:

HTML5, CSS3, JavaScript

Responsive design, Flexbox/Grid

Vanilla JS for interactivity and form handling

Features:

Smooth scrolling navigation

Interactive UI components (tabs, accordion, menu)

Contact form with server submission

Fully responsive layout for mobile and desktop
